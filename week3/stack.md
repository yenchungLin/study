# stack
## 原理說明：    
先進後出的資料型結構，例如自助餐盤一樣，餐盤疊起來後，使用時會拿最上端的那個。   
五個基本工作定義：      
 *   create : 建立一個空堆
 *   push : 存放頂端資料，回傳新堆疊
 *   pop : 刪除頂端資料，並回傳新堆疊
 *   isEmpty : 判斷堆疊是否為空堆疊
 *   full : 判斷堆疊是否已滿   
應用：   
* DFS
