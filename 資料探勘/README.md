## 資料分類(Classification)
1. 定義：   
>* 將一個物件指定預先定義的分類   
>* 建立一個學習函數(分類模型)，將每個屬性集(x)對應到一個已定義的類別(y)   
2. 功能：   
>* 描述已知類別的特徵，用於解釋與區分不同的類別   
>* 預測未知類別的物件，用於判斷物件所屬類別   
3. 
>* K最鄰近法(KNN)
>* 決策樹
